# Info Kelas - 2IA15

**Info Kelas** adalah aplikasi manajemen kelas kuliah pribadi berbasis web. Aplikasi ini memungkinkan mahasiswa untuk memantau jadwal kuliah, tugas, dan mata kuliah secara praktis melalui dashboard yang interaktif.  

Dibangun menggunakan **Next.js**, dengan penyimpanan data menggunakan **MongoDB** dan **SQLite3**.

---

## Fitur Utama

- **Dashboard Interaktif**  
  Menampilkan ringkasan tugas, jumlah mata kuliah, dan kelas hari ini.
- **Manajemen Tugas**  
  Tambah, hapus, dan tandai tugas selesai.
- **Jadwal Kuliah**  
  Menampilkan jadwal kelas setiap hari.
- **Info Mata Kuliah**  
  Daftar lengkap mata kuliah yang diikuti.
- **Pemberitahuan & Pengumuman**  
  Fitur marquee & card untuk menampilkan informasi penting.
- **Autentikasi Sederhana**  
  Login untuk mengakses fitur tertentu (direncanakan).

---

## Teknologi

- **Frontend & Server**: [Next.js](https://nextjs.org/) (React + Server Components)
- **Database**:
  - [MongoDB](https://www.mongodb.com/) untuk menyimpan tugas dan jadwal secara fleksibel.
  - [SQLite3](https://www.sqlite.org/index.html) untuk data lokal atau backup ringan.
- **UI & Styling**: Tailwind CSS
- **Ikon**: [Lucide React](https://lucide.dev/)
